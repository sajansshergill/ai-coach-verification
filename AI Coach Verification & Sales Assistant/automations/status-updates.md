## Automation Setup

### Using Zapier/Make.com

**Workflow Example:**

1. **Trigger:** New coach signup OR status change in database
2. **Action:** Send email via SendGrid/Mailgun
3. **Condition:** Check which email template to send
4. **Delay:** Add delays for reminder emails (24-48 hours)
5. **Filter:** Don't send if coach already completed/approved

**Tools Needed:**
- Zapier or Make.com account
- Email service (SendGrid, Mailgun, Postmark)
- Database/webhook integration
- Template variables

---

### Using Native Platform

**If building custom:**

1. **Set up email service** (SendGrid, Mailgun, etc.)
2. **Create email templates** (use templates above)
3. **Set up triggers** (database events, status changes)
4. **Configure delays** (for reminder emails)
5. **Test thoroughly** (test all email types)

## Personalization Variables

**Use these variables in templates:**

- `[Coach Name]` - Coach's first name
- `[Platform Name]` - Your platform name
- `[AI Assistant Name]` - AI assistant name (e.g., "Verification Assistant")
- `[Tier]` - Verification tier (Basic/Pro/Premium)
- `[Link to continue verification]` - Direct link to verification flow
- `[Link to profile]` - Link to coach's profile
- `[Link to dashboard]` - Link to coach dashboard
- `[Support email]` - Support email address
- `[Admin Name]` - Admin name (for manual emails)
